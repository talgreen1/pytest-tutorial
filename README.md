# Pytest 
Run all tests:  
```pytest```

Run with verbos:  
```pytest -v```

Run tests in parallel:
```pytest -n 3```


##Custom Markers
Run custom marker:  
```
pytest -m smoke
pytest -m "body or engine"
pytest -m "not entertainment"
 ```

Do not capture prints:
```pytest -s```
